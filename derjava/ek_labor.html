Эта страница разрабатывается в рамках проектирования экономической модели народовластия для наглядного примера как это может работать. Вводимые данные также будут использоваться в потребительских кооперативах.
Для кооперативов и народной экономики вопрос справедливой оценки труда имеет крайне важное значение. Предлагаем проставить значения ставок заработной платы в условных единицах. Ваше голосование будет ориентиром для кооперативов и приниматься за основу при распределении производимой продукции. Для каждого кооператива будет дана отдельная таблица этих показателей составленнай на основе данных кооператоров.
При предложении зарплат предполагайте, что это базовая ставка, которая учитывает необходимый уровень образования. Повышающие коэфициенты предлагается установить исходя из времени обучения тем или иным специальностям.

Уровень компетенции коэффициент
1. начальное обазование 5 лет 1
2. среднее обазование 5 лет или работа по специальности без диплома 2
3. высшее образование или работа по специальности без диплома 3 +0.2 за каждый год
4. стажировка 5 лет 4 +0.2 за каждый год
5. эксперт 5
