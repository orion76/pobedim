Как организовать референдум.
  1. Создать группу Референдум в при совет-вече (в объединении есть кнопка "Создать референдум")
  2. Подготовить список проблем
  3. По итогам голосований сформировать рабочи группы для оформления документов

Создание группы референдума

Референдум создаюётся как группа при вече для формализованной работы. То есть, чтобы оформить народное решение по отдельным вопросам по действующим юридическим нормам силами инициативных групп.

Список проблем

Список проблем - это перечень опросов (голосований). Каждый опрос - это проблема, которую требуется решить на референдуме. В форме группы референдума, нажмите кнопку "Добавить в список проблему для решения". Откроется форма "Создание голосования". В поле "Название" кратко сформулируйте суть проблемы. После создания голосования (добавления в список проблем для решения) эта проблема появится в списке голосований группы.

Голосование ( опрос по проблеме ) предназначен для выбора сути вопроса для референдума с предполагаемым ответом "ДА". На данном этапе не требуется точность формулировки, которая появится на формальных бланках проводимого референдума, важна СУТЬ выносимого решения. Голос за решение по сути, подразумевает также утвердительный ответ и по существу опроса. После того как будет проведено голосования, формулировкой вопроса займётся рабочая группа. Данный процесс описан ниже.

Когда количество проголосовавших по опросу (проблеме) преодолеет формальный порог для референдума, любой из активистов нажимает кнопку "Создать рабочую группу" и, таким образом, инициирует подготовку к проведению референдума с выполнением юридических формальностей.

После нажатия кнопки появится форма, в которой требуется ввести название группы. Название группы должно отражать суть будущего опроса на референдуме.

Работа в рабочей группе ничем не отличается от аналогичных групп. В рамках группы проводится обсуждение и вырабатывается формулировка вопроса референдума. Координатор в форме редактирования группы создаёт итоговое голосование по формулировке текста вопроса на референдуме. Содержание формулировки не должна изменять сути решения проблемы (противоречить цели рабочей группы) и должна соответствовать ответу "ДА".
